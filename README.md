# 思想・政治信条が報道を歪める現状に関する調査報告書

## 概要

本リポジトリは、2026年3月16日に発生した**沖縄県辺野古沖小型船舶転覆事故**と、2026年4月に発生した**京都小学生死体遺棄事件**における報道実態を比較分析し、思想・政治信条がいかにして報道を歪め、真実を覆い隠すかを明らかにする調査レポートです。

## 📋 主な内容

- **辺野古ボート転覆事故の全体像**
  - 事故発生の経緯と安全管理の欠如
  - 犠牲者・武石知華さんの実像と政治的レッテル貼り
  - 教育機関のガバナンス不全
  - 運航団体の法的問題と不誠実な対応

- **京都小学生死体遺棄事件における報道の歪み**
  - SNSデマの拡散メカニズム
  - 排外主義・外国人差別に基づく歪み
  - 台湾メディアの誤報

- **両事件の比較分析**
  - 思想的バイアスの方向性の違い
  - 「大衆の差別思想」vs「メディアの政治思想」
  - 「報道しない自由」の実態

- **教育と政治活動の境界線**
  - 修学旅行を利用した政治活動資金源の問題
  - 「平和教育」の名を借りた政治活動

## 🎯 本レポートの目的

1. **真実の記録**: 思想や政治信条によって歪められた事実を正確に記録する
2. **被害者の尊厳**: 亡くなった武石知華さんの本当の姿を伝える
3. **構造的問題の指摘**: 教育現場とメディアにおける構造的欠陥を明らかにする
4. **再発防止**: 同様の悲劇を繰り返さないための提言を行う

## 🚨 重要な問題提起

### 辺野古事故における「報道しない自由」

- **知床遊覧船事故との比較**: 同様の海難事故でありながら、報道量に著しい差
- **BPOへの苦情殺到**: 視聴者から「報道回数が少なすぎる」との批判
- **安全管理の追及不足**: 運航団体や学校の責任が十分に追及されていない

### 修学旅行利権の実態

- 年間約2,000校の修学旅行生が沖縄を訪問
- 反基地活動の重要な資金源となっている可能性
- 「ボランティア」という隠れ蓑での実質的な営業行為

### 教育の政治的中立性への疑問

- 特定の政治的立場に偏った「平和学習」
- 学習指導要領・教育基本法への抵触の可能性
- 多様な視点を提供しない一方的な教育内容
